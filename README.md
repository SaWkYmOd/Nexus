🚀 Overview
The Nexus Platform is the ultimate B2B2C Android TV App Store Builder. It serves as a seamless bridge between App Creators and TV End-Users.

Instead of forcing users to navigate messy URLs or complex sideloading instructions, Creators can build a fully managed storefront on the web, upload their apps, and deliver them directly to their users' TVs using a simple 9-digit Connect ID.

🛠️ How It Works
Build on the Web: Creators log into the Creator Dashboard to upload logos, set brand colors, and add APK download links into a clean database.

Install the TV App: End-users download the secure, lightweight Nexus Platform app directly to their Android TV, Firestick, or set-top box.

Enter Connect ID: Users enter the Creator's unique 9-digit Connect ID, and the app instantly morphs into a fully branded, customized app store.

✨ Key Features
For Creators (Web Dashboard)
App Management: Easily add apps with icons, screenshots, version numbers, and package IDs.

Categories & Organization: Sort apps into custom or default categories (e.g., Entertainment, Utilities, IPTV).

Parental Controls: Includes an auto-restricted XXX category to lock adult content behind a master 4-digit PIN.

Maintenance Mode: Hide specific apps from the storefront temporarily with a single toggle.

Premium Branding: Upgrade via Stripe to unlock custom Store Banners, Login Icons, and dynamic Theme Accent Colors.

For End-Users (Android TV App)
Built-in Downloader: Features a fully integrated, hardware-accelerated web browser and download manager optimized for D-Pad remotes.

Smart File Detection: Instantly detects direct APK links (like Dropbox dl=1 or Catbox) and downloads them directly.

Advanced App Handling: Detects installed app versions and seamlessly switches the UI between "Download", "Update", or "Open".

Cloud Backup: Premium users can save their connected stores and preferences to the cloud and restore them on any TV.

File Manager: Built-in tool to install or delete leftover APKs to save storage space.
